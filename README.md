# Speed Typing Game - Frontend

Modern, responsive frontend for the real-time multiplayer typing game.

## Features
- Clean dark theme UI
- Real-time leaderboard with player avatars
- Face card avatars for each player
- Responsive design (desktop, tablet, mobile)
- Live submission feed
- Visual countdown timer
- Socket.IO client integration

## Structure

```
frontend/
├── static/
│   ├── style.css      # Main stylesheet
│   ├── main.js        # Home page logic
│   └── game.js        # Game room logic
└── templates/
    ├── index.html     # Home page
    └── game.html      # Game room page
```

## Tech Stack
- HTML5
- CSS3 (Custom styling, no frameworks)
- JavaScript (ES6+)
- Socket.IO Client 4.5.4

## Design Features
- Professional dark theme (#0a0f1e background)
- Color-coded player avatars
- Sticky left sidebar leaderboard
- Smooth animations and transitions
- Mobile-friendly submit button
