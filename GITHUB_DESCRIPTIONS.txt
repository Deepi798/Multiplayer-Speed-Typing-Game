# Speed Typing Game

A professional real-time multiplayer typing game where players compete to type sentences as fast and accurately as possible.

## 🎮 Features

- **Real-time Multiplayer** - Compete with friends using WebSockets
- **Solo Play Mode** - Practice alone and improve your speed
- **Progressive Difficulty** - 15 rounds: Easy → Medium → Hard
- **Live Leaderboard** - See rankings update in real-time
- **Room System** - Create/join rooms with unique codes
- **Mobile Friendly** - Works on desktop, tablet, and phone
- **Professional UI** - Modern dark theme with clean design
- **Smart Scoring** - Points based on speed and accuracy

## 🚀 Quick Start

### Local Development
```bash
# Clone repository
git clone https://github.com/YOUR_USERNAME/speed-typing-game.git
cd speed-typing-game

# Install dependencies
pip install -r requirements.txt

# Run application
python app.py

# Open browser
http://localhost:5000
```

## 🎯 How to Play

1. **Create Room** - Enter your name and create a new game room
2. **Share Code** - Share the room code with friends
3. **Start Game** - Begin when all players are ready
4. **Type Fast** - Type the displayed sentence as quickly as possible
5. **Earn Points** - Faster typing = more points
6. **Win** - Highest score after 15 rounds wins!

## 🛠️ Tech Stack

- **Backend:** Flask, Flask-SocketIO, Python
- **Frontend:** HTML5, CSS3, JavaScript
- **Real-time:** Socket.IO, WebSockets
- **Deployment:** Gunicorn, Eventlet
