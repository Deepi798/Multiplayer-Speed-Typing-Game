========================================
HOW TO GET A PUBLIC LINK (NGROK)
========================================

QUICK TEMPORARY LINK (Good for testing):

1. Download ngrok: https://ngrok.com/download
2. Extract ngrok.exe to your project folder
3. Open Command Prompt in your project folder
4. Run: python app.py
5. Open ANOTHER Command Prompt
6. Run: ngrok http 5000
7. You'll see a public URL like: https://abc123.ngrok.io
8. Share this URL with anyone!

Note: Free ngrok links expire when you close it.

========================================
PERMANENT FREE HOSTING (RENDER)
========================================

1. Install Git: https://git-scm.com/download/win

2. Create GitHub account: https://github.com

3. In your project folder, run these commands:
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   
4. Create new repository on GitHub

5. Push code:
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
   git push -u origin main

6. Go to https://render.com and sign up

7. Click "New +" → "Web Service"

8. Connect your GitHub repository

9. Settings:
   - Build Command: pip install -r requirements.txt
   - Start Command: gunicorn --worker-class eventlet -w 1 app:app

10. Click "Create Web Service"

11. Wait 5-10 minutes for deployment

12. You'll get a permanent URL like:
    https://your-app-name.onrender.com

========================================
REQUIRED FILE UPDATES FOR DEPLOYMENT
========================================

You need to update requirements.txt:

Flask==3.0.0
Flask-SocketIO==5.3.5
python-socketio==5.10.0
gunicorn==21.2.0
eventlet==0.33.3
